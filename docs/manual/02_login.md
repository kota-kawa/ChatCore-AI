# ログイン・アカウント作成

Chat-Core-AI では 3 種類のログイン方法を使用できます。パスワードは不要です。

## メールコード認証（メイン手順）

1. `/login` にアクセスする
2. メールアドレスを入力して「コードを送信」をクリック
3. 受信したメールに記載の **6桁の確認コード** を入力する
4. 「確認」をクリックしてログイン完了

> メールが届かない場合は迷惑メールフォルダを確認してください。

## Google アカウントでログイン

1. ログインページで「Google でログイン」ボタンをクリック
2. Google の認証画面でアカウントを選択・許可
3. 自動的にチャットページへ移動

## Passkey（生体認証）でログイン

Passkey を事前に登録している場合、メールコード不要でログインできます。

1. ログインページでメールアドレスを入力
2. 「Passkey でログイン」をクリック
3. ブラウザの生体認証ダイアログ（指紋・顔認証など）で認証
4. ログイン完了

Passkey の登録方法は「設定 > セキュリティ」を参照してください。

## ログアウト

1. 画面右上のユーザーアイコンをクリック
2. ドロップダウンメニューから「ログアウト」を選択

## セッションの有効期限

ログイン状態は **30 日間** 維持されます。期限が切れると自動的にログアウトされます。
