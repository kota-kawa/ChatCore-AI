# 共有機能まとめ

Chat-Core-AI では、チャット・メモ・プロンプトの 3 種類のコンテンツを共有できます。

## 共有できるコンテンツ一覧

| 共有対象 | 手順 | 共有先 |
|---------|------|--------|
| チャット履歴 | チャット画面の「共有」ボタン | URL リンク / SNS |
| メモ | メモ詳細画面の「共有」ボタン | URL リンク / SNS |
| プロンプト | プロンプト詳細の「共有」ボタン | URL リンク / SNS |

## SNS 共有対応

- Twitter（X）
- LINE
- Facebook
- ネイティブシェア（スマートフォン）

## 共有時の注意

- 共有リンクを知っている人は **誰でも** コンテンツを閲覧できます
- 共有を停止したい場合は、元のコンテンツを削除してください
- チャットの共有リンクには有効期限が設定される場合があります
