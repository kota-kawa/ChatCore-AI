# 設定画面

## 設定画面を開く

画面右上のユーザーアイコンをクリック → 「設定」を選択、または `/settings` に直接アクセスします。

## プロフィール設定

「プロフィール設定」タブで以下を編集できます。

| 項目 | 説明 |
|------|------|
| ユーザー名 | 表示名 |
| メールアドレス | ログイン用メール |
| 自己紹介 | プロフィール説明文 |
| プロフィール画像 | アイコン画像（クリックしてアップロード） |
| AI に伝えておきたい情報 | AI が常に参照するパーソナライズ情報 |

変更後は「変更を保存」ボタンをクリックします。

### 「AI に伝えておきたい情報」の活用例

この欄に入力した内容は、すべてのチャットで AI に自動的に伝わります。

- 「私はエンジニアです。技術的な説明を詳しくしてください」
- 「回答は必ず日本語でお願いします」
- 「箇条書きよりも文章形式で回答してほしい」

## 外観（テーマ）設定

| テーマ | 説明 |
|--------|------|
| ライト | 白を基調とした明るいデザイン |
| ダーク | 黒を基調とした暗いデザイン |
| システムに合わせる | OS のダークモード設定に自動追従 |

設定はブラウザのローカルに保存されます（端末ごとに個別設定が必要です）。

## Passkey（生体認証）の管理

「セキュリティ」タブで Passkey を管理できます。

### Passkey を登録する

1. 「この端末に Passkey を追加」をクリック
2. ブラウザの認証ダイアログ（指紋・顔認証）で認証
3. 登録完了。リストに追加される

登録後は次回からメールコードなしでログインできます。

### Passkey を削除する

1. 「セキュリティ」タブで対象 Passkey を確認
2. 「削除」ボタンをクリック

> すべての Passkey を削除した場合、次回ログインはメールコード認証になります。

### Passkey が使えない場合の確認事項

- ブラウザが最新版であること
- 端末が生体認証（指紋・顔認証）に対応していること
- HTTPS 接続であること

## プロンプト管理

「プロンプト管理」タブで自分が投稿したプロンプトを一覧表示・編集・削除できます。
