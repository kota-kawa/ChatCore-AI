# セキュリティ・プライバシー

## 認証セキュリティ

| 機能 | 説明 |
|------|------|
| **パスワードレス認証** | メールコードまたは Passkey によるログイン。パスワード漏洩リスクがない |
| **Passkey（WebAuthn）** | 生体認証を使った暗号化ログイン。フィッシング攻撃に強い |
| **セッション管理** | 30 日間有効なセキュアセッション。HTTPS 通信時は Secure Cookie を使用 |

## 通信セキュリティ

| 機能 | 説明 |
|------|------|
| **CSRF 保護** | 不正なリクエストを防ぐトークン検証（すべての更新操作に適用） |
| **セッション固定攻撃対策** | ログイン時にセッション ID を再発行 |

## AI に送信するデータの保護

**秘密情報の自動伏せ字化**: チャットで API キー・OAuth トークン・パスワードなどの秘密情報を入力した場合、AI に送信される前に自動的に伏せ字化（`****`）されます。

## 共有コンテンツのプライバシー

- チャット・メモ・プロンプトの共有リンクは、知っている人なら誰でもアクセスできます
- 公開したくない情報は共有しないようにしてください
- 共有を停止するにはコンテンツを削除してください
