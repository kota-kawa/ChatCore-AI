# プロンプト共有機能

コミュニティのユーザーが投稿したプロンプトを閲覧・利用・共有できます。自分のプロンプトを投稿してシェアすることもできます。

## プロンプトを探す

1. `/prompt_share` にアクセス
2. カテゴリフィルターで絞り込む
3. 検索ボックスにキーワードを入力して `Enter`
4. プロンプトカードをクリックして詳細を表示

## プロンプトを投稿する

1. プロンプト共有ページの「プロンプトを投稿」ボタンをクリック
2. 投稿フォームに以下を入力する：

| フィールド | 説明 | 最大長 | 必須 |
|-----------|------|--------|------|
| タイトル | プロンプトの名前 | 256 文字 | ✓ |
| カテゴリ | 分類 | 256 文字 | ✓ |
| プロンプト本体 | 実際のプロンプト | 4,000 文字 | ✓ |
| 投稿者名 | 表示名 | 256 文字 | - |
| 使用 AI モデル | 動作確認済みモデル | 256 文字 | - |
| 入力例 | サンプル入力 | 4,000 文字 | - |
| 出力例 | 期待する出力例 | 4,000 文字 | - |
| 参照画像 | 補足画像 | 5MB 以内 | - |
| SKILL定義 (Markdown) | Claude Code / Codex CLI 向けのSKILL本文 | 30,000 文字 | SKILL投稿時のみ ✓ |
| 追加 Python スクリプト | SKILLで使う補助スクリプト | 30,000 文字 | - |

3. 投稿タイプで「通常プロンプト」「画像生成プロンプト」「SKILL」を選択できる
4. 「AI 補助」ボタンでタイトルや説明を自動生成できる
5. 「投稿」をクリック

## プロンプトを共有する

1. 対象プロンプトの詳細を開く
2. 「共有」ボタンをクリック
3. 共有 URL をコピー、または SNS（Twitter・LINE・Facebook）でシェア

## コメント・いいね・チャットで使う

| 操作 | アイコン | 説明 |
|------|--------|------|
| コメント | 💬 | 詳細モーダルから感想や改善案を投稿 |
| いいね | ♡ | 気に入ったプロンプトに評価をつけ、設定画面から後で確認 |
| チャットで使う | ＋ | 自分のチャット用タスクとして追加 |

## コメント運用ルール（荒れ対策）

- コメントはログインユーザーのみ投稿できます
- 短時間の連続投稿・重複投稿は自動で制限されます
- URLを大量に含むコメントは投稿できません
- コメントは投稿者本人・対象プロンプトの投稿者・管理者が削除できます
- 通報が一定数集まったコメントは自動で非表示になります

## 自分の投稿を管理する

1. `/settings` → 「投稿したプロンプト」タブ
2. 投稿済みプロンプト一覧が表示される
3. 編集・削除が可能
