# プロンプト共有機能

コミュニティのユーザーが投稿したプロンプトを閲覧・利用・共有できます。自分のプロンプトを投稿してシェアすることもできます。

## プロンプトを探す

1. `/prompt_share` にアクセス
2. カテゴリフィルターで絞り込む
3. 検索ボックスにキーワードを入力して `Enter`
4. プロンプトカードをクリックして詳細を表示

## プロンプトを投稿する

1. プロンプト共有ページの「プロンプトを投稿」ボタンをクリック
2. 投稿フォームに以下を入力する：

| フィールド | 説明 | 最大長 | 必須 |
|-----------|------|--------|------|
| タイトル | プロンプトの名前 | 256 文字 | ✓ |
| カテゴリ | 分類 | 256 文字 | ✓ |
| プロンプト本体 | 実際のプロンプト | 4,000 文字 | ✓ |
| 投稿者名 | 表示名 | 256 文字 | - |
| 使用 AI モデル | 動作確認済みモデル | 256 文字 | - |
| 入力例 | サンプル入力 | 4,000 文字 | - |
| 出力例 | 期待する出力例 | 4,000 文字 | - |
| 参照画像 | 補足画像 | 5MB 以内 | - |

3. 「AI 補助」ボタンでタイトルや説明を自動生成できる
4. 「投稿」をクリック

## プロンプトを共有する

1. 対象プロンプトの詳細を開く
2. 「共有」ボタンをクリック
3. 共有 URL をコピー、または SNS（Twitter・LINE・Facebook）でシェア

## いいね・ブックマーク・リスト保存

| 操作 | アイコン | 説明 |
|------|--------|------|
| いいね | ♡ | 気に入ったプロンプトに評価をつける |
| ブックマーク | ★ | 後で見返したいプロンプトを保存 |
| リストに追加 | 📋 | 自分のリストに整理して保存 |

## 自分の投稿を管理する

1. `/settings` → 「プロンプト管理」タブ
2. 投稿済みプロンプト一覧が表示される
3. 編集・削除が可能
