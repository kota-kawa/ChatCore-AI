# タスク（プロンプトテンプレート）

タスクは、よく使うプロンプトをテンプレートとして登録・管理する機能です。毎回同じ指示を入力する手間を省けます。

## タスク一覧を見る

チャットページの「タスク」パネルに登録済みタスクが表示されます。

## タスクを使う

1. タスク一覧から使いたいタスクをクリック
2. タスクのプロンプトテンプレートがチャット入力欄に展開される
3. 必要に応じて編集してから送信

## タスクを作成する

1. チャット画面の「新規プロンプト」ボタンをクリック
2. 「新規プロンプト作成」モーダルが開く
3. 以下のフィールドを入力する：

| フィールド | 説明 | 必須 |
|-----------|------|------|
| タイトル | タスクの名前 | ✓ |
| プロンプトテンプレート | AI に送るプロンプト本体 | ✓ |
| 応答ルール | AI の応答形式・スタイルの指定 | - |
| 出力形式 | 出力のひな型（Markdown や表形式など） | - |
| 入力例 | サンプル入力 | - |
| 出力例 | 期待する出力例 | - |

4. 「保存」をクリック

## タスクを編集する

1. タスク一覧で対象タスクをクリックして詳細を開く
2. 「編集」ボタンをクリック
3. 内容を変更して「保存」

## タスクを削除する

1. タスク詳細モーダルを開く
2. 「削除」ボタンをクリック
3. 確認ダイアログで「削除」を選択

## タスクを並び替える

タスク一覧でドラッグ＆ドロップして順序を変更できます。よく使うタスクを上位に移動すると便利です。
