# メモ機能

AI の回答から生成したメモを保存・管理・共有できます。重要な情報や後で参照したい内容を整理するのに便利です。

## メモを作成する

チャット画面で保存したいメッセージの「メモに保存」ボタンをクリックします。

## メモ一覧を見る

`/memo` にアクセスすると保存済みメモが一覧表示されます。

## メモにタグを付ける

メモ詳細画面からタグを追加・編集できます。タグで分類することで後から検索しやすくなります。

## メモを共有する

1. 対象メモの「共有」ボタンをクリック
2. 共有 URL を取得
3. SNS（Twitter・LINE・Facebook）でシェア、またはリンクをコピーして共有

共有リンクを知っている人は誰でもメモを閲覧できます。
