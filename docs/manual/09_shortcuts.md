# キーボードショートカット

## チャット画面

| キー | 動作 |
|------|------|
| `Enter` | メッセージを送信 |
| `Escape` | 開いているモーダルを閉じる |
| `Tab` | モーダル内でフォーカスを次の要素に移動 |
| `Shift + Tab` | モーダル内でフォーカスを前の要素に移動 |

## プロンプト共有ページ

| キー | 動作 |
|------|------|
| `Enter` | 検索を実行 |
| `Escape` | モーダルを閉じる |
| `Tab` | モーダル内フォーカス移動 |

## 設定画面

| キー | 動作 |
|------|------|
| `Escape` | 編集モーダルを閉じる |
