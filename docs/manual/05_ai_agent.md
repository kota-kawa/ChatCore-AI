# AI エージェント機能

画面左下の **AI エージェントボタン**（丸いアイコン）は、すべてのページで表示される補助 AI 機能です。現在操作中のページに関する質問や作業の補助をリクエストできます。
ログイン/登録画面では、認証操作の誤実行を避けるため表示されません。

## 使い方

1. 画面左下の AI エージェントボタンをクリック
2. チャットパネルが開く
3. 質問や依頼を入力して送信
4. AI が回答する

## できること

| 用途 | 例 |
|------|-----|
| 操作方法の質問 | 「このページでプロンプトを共有するにはどうすればいいですか？」 |
| コンテンツの要約 | 「現在のチャット内容を3行でまとめてください」 |
| プロンプト改善 | 「このプロンプトをより具体的にしてほしい」 |
| タスク作成補助 | 「翻訳タスクのテンプレートを作ってください」 |
| 機能の説明 | 「Passkey とは何ですか？」 |
| 画面操作の補助 | 「プロンプト共有を開いてメール返信を検索して」 |

## 操作を実行するときの確認

AI が操作手順を作成した場合、内容を確認してから **実行** ボタンを押します。
保存、送信、外部認証につながる可能性がある操作では、実行前に確認ダイアログが表示されます。

## ページをまたいで利用可能

AI エージェントボタンはチャットページ以外（プロンプト共有・メモ・設定など）でも表示されます。どこからでもすぐに AI に質問できます。
